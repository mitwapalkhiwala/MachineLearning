This project predicts the breast cancer based on various parameters available in the dataset. 

The dataset can be found in bc.csv

Logistic Regression.ipynb file is the python code to predict the breast cancer.

Logistic Regression.r file is the R code to predict the breast cancer.

Naive Bayes Classifier to predict breast cancer.ipynb is the python code that defines the Naive Bayes class from scratch to predict breast cancer.

SVM to predict breast cancer.ipynb contains the python code for SVM classifier to predict the breast cancer.
